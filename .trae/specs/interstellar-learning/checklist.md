1. [ ] 用户打开应用 → 显示首页仪表盘
2. [ ] 用户记录心情 → 心情保存并显示
3. [ ] 用户生成计划 → 根据心情生成个性化计划
4. [ ] 用户打开番茄钟 → 显示计划中的任务
5. [ ] 用户选择任务开始番茄 → 番茄钟运行

9. [ ] 用户查看知识星球 → 显示收藏的知识点
10. [ ] 用户点击链接 → 跳转到相关知识点
